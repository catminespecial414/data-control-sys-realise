# 系统设计文档

## 概述

本系统旨在实现农业数据控制，包括传感器数据采集、设备控制、网络通信、数据库管理和自动化控制逻辑。

## 模块设计

### 1. Backend
- 主程序入口和调度器。

### 2. Sensors
- 包括温湿度传感器、土壤湿度传感器和气压传感器。

### 3. Devices
- 控制继电器、LED和蜂鸣器。

### 4. Network
- 提供HTTP通信功能。

### 5. Control
- 实现自动灌溉逻辑。

### 6. Database
- 使用SQLite进行数据存储。

### 7. AI
- 提供数据分析和预测功能。

### 8. Web
- 提供基于Flask的Web界面。

## 数据流

1. 传感器采集数据。
2. 数据存储到SQLite数据库。
3. 控制逻辑根据数据触发设备动作。
4. 数据通过网络上传到云端。
5. Web界面展示数据和控制设备。

## 未来扩展

- 增加更多传感器支持。
- 集成更多AI分析功能。
- 支持更多网络协议。
