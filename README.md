# Farm IoT System

**农田数据采集控制系统（基于树莓派 + WiFi）**

## 1 项目简介

本项目是一个基于 **树莓派（Raspberry Pi）** 的农田数据采集与自动控制系统。
系统通过多种传感器采集农田环境数据，并通过 WiFi 网络进行数据传输，实现：

* 农田环境数据监测
* 自动灌溉控制
* 数据存储与分析
* Web可视化监控
* AI预测分析

系统采用模块化设计，方便扩展新的传感器或控制设备。

---

# 2 系统功能

本系统主要实现以下功能：

### 1 环境数据采集

系统通过传感器采集农田环境数据：

* 温度
* 湿度

---

### 2 自动控制系统

系统根据采集到的数据进行判断，例如：

```
当土壤湿度 < 30%
自动开启灌溉系统
```

通过继电器控制电机。

---

### 3 数据存储

系统使用 **MySQL数据库** 存储历史数据：

* 时间
* 温度
* 湿度

用于后续数据分析。

---

### 4 Web监控界面

系统提供 Web 页面，可以查看：

* 实时环境数据
* 历史数据
* 系统运行状态

通过浏览器即可访问：

```
http://树莓派IP:5000
```

---

### 5 AI数据分析

AI模块通过分析历史数据实现：

* 土壤湿度预测
* 灌溉需求预测
* 农田环境趋势分析（图片识别）

---

---

# 4 各模块说明

### backend

系统主程序模块，负责系统整体运行调度：

* 传感器数据采集
* 调用控制逻辑
* 数据存储
* 数据上传

---

### sensors

传感器数据采集模块

* 温湿度传感器
* 摄像头

---

### devices

设备控制模块

* 触控屏
* 蜂鸣器
* 直流电机

---

### network

网络通信模块，负责通过 WiFi 进行数据传输

* HTTP数据上传

---

### control

自动控制模块，实现农田自动控制逻辑，例如：

* 自动灌溉
* 报警控制

---

### database

数据库模块，负责数据存储与查询。

系统使用 **MySQL数据库**。

---

### ai

AI数据分析模块，负责：

* 数据预测
* 农田环境趋势分析
* 智能控制优化

---

### web

Web监控系统模块，提供可视化界面：

* 实时数据展示
* 历史数据查看
* 系统状态监控

---

### docs

项目文档，包括：

* 系统设计说明
* 硬件连接图
* 项目报告
